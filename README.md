# Fake Identity Generator [WIP]
Bash script for generating unique identities for assisting purposes like OSINT for example.

## Getting Started

Clone the repo and do `chmod a+r *` and `chmod +x ident_gen.sh`

### Prerequisites

Developed with: 
* GNU bash 5.0.11
* mawk 1.3.3
